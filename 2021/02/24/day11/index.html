File

  • 传入路径时可以是绝对路径或相对路径，相对路径以根目录开头

  • Windows平台使用 \ 作为路径分隔符，即使用 \\ 表示 一个 \ ; Linux平台使用 / 作为路径分隔符

  • getPath()返回构造方法传入的路径；getAbsolutePath()返回绝对路径；getCanonicalPath()返回规范路径

文件和目录

​ File对象既可以表示文件，也可以表示目录，构造File对象时不会对磁盘进行操作，只有调用方法时才会对磁盘进行操作。

1. isFile(), isDirectory()：判断是否是存在的文件，目录
2. 一个文件对象，还有canRead(), canWrite(), canExecute(), length()等方法，表示是否可读，是否可写，是否可执行，文件字节大小

创建和删除文件

​ 当File对象表示文件时，使用createNewFile()**创建一个新文件，delete()删除文件。如果要创建临时文件使用createTempFile()，deleteOnExit()**在JVM退出时删除该文件。

遍历文件和目录

​ 当File对象表示目录时，可以使用list()和listFiles()列出目录下的文件和子目录:

  • boolean mkdir()：创建当前File对象表示的目录
  • boolean mkdirs()：创建当前File对象表示的目录，并在必要时将不存在的父目录创建
  • boolean delete()：删除当前File对象表示的目录，必须为空才能删除成功

Path

​ Path对象主要操作文件路径：

InputStream

​ InputStream是所有输入流的抽象类，最重要的方法是int read()。该方法会读取输入流的下一个字节，并返回字节表示的int值，读到末尾返回-1

  • 使用输入流读取数据后，最后要使用close()关闭数据流
  • 读取数据时可能会出现文件不存在等情况，因此需要抛出IOException。为了在出错时可以释放资源，使用try…finally语句，或者try(resource)

缓冲

​ 利用缓冲区一次读取多个字节数据，int read(byte[])，读取若干字节并填充到byte[]数组，返回读取的字节数；int read(byte[],int off, int len)，指定byte[]数组的偏移量和最大填充数

  • 使用read方法进行数据读取时，是阻塞的，即数据读取不完成就无法继续执行代码

实现类

​ FileInputStream 可以从文件中获取输入流，ByteArrayInputStream可以在内存中模拟一个InputStream。

OutputStream

​ OutputStream是所有输出流的超类，也是抽象类，最重要的方法时void write(int b)。该方法会写入一个字节的数据，相当于b&0xff

  • OutputStream除了close()方法外，还有一个flush()方法，作用是将缓冲区的内容真正输出到目的地

    OutputStream实现类

​ FileOutputStream可以从文件中获得输出流，ByteArrayOutputStream可以在内存中模拟一个OutputStream

Reader

​ Reader是java中另一输入流接口，和InputStream不同的是，Reader读取的是字符流，即以char为单位读取。

​ java.io.Reader是所有字符输入类的超类，但不是接口。最主要的方法是int read()，返回字符表示的int值，读到末尾返回-1

FileReader

​ 是Reader的子类，可以打开文件并获取Reader：

  • 为了避免乱码问题，在创建FileReader的时候要指定编码
  • 和Inputstream相同，使用Reader时也需要使用try…finally语句，Reader也提供了一次性读取若干字符的方法int read(char[] c)

CharArrayReader

​ 它可以在内存中模拟一个Reader，和ByteArrayInputStream作用类似，将char[]作为数据源。StringReader是把String作为数据源。

  • Reader其实是一个基于InputStream的byte到char的转换器，因此可以把任何InputStream转化为Reader,使用InputStreamReader

Writer

​ Writer是带编码转换器的OutputStream，他把char转换为byte输出。Writer是所有字符流输出的超类，他提供的方法：

  1. void write(int c):写入一个字符
  2. void write(char[] c):吸入字符数组的所有字符
  3. void write(String s):写入String表示的所有字符

FileWriter

  • 可以使用OutputStreamWriter把任意的OutputStream转换为Writer

PrintStream和PrintWriter

​ PrintStream是一种FilterOutputStream,他在OutputStream的接口上额外提供了数据写入的方法：print(int/boolean/String/Object)。System.out.println()实际上就是使用PrintStream打印各种数据，System.err是系统默认的标准错误输出

​ PrintWriter扩展了Writer接口，该方法最终输出的是char数据。

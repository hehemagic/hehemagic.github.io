java常用类

发表于 2021-09-13 更新于 2021-06-02

String

​ 字符串String内部是通过一个char[]数组表示的，java中“…”代表字符串，’.’代表字符char。

  • java字符串不可变，因为其内部字段为private final char[]以及没有任何修改char[]的方法实现的
    • 修改字符串时，并不是修改了原来的字符串，而是在常量池中新创建了一个字符串
    • 只有创建常量池中不存在的字符串才会创建新的字符串
    • 使用new 创建String对象时，会同时在堆和常量池中都创建对象，栈中的引用指向堆中的对象
  • 比较String是否相同时要使用equals()方法而不能使用==
  • java编译器在初期会将所有的字符串放进常量池，相同的字符串合并成一个对象

String 方法

  1. **equalsIgnoreCase()**：比较两个字符串，忽略大小写

  2. contains(Char Sequence):判断字符串是否包含指定字符串

  3. **indexof()，lastIndexOf()**：获得指定子串的索引

  4. **startsWith()，endsWith()**：判断字符串是否以指定子串开头，结尾

  5. **subString()**：获得指定索引的子串

  6. **trim()**：去除字符串首尾空白字符（空格，\t，\r，\n），返回新的字符串

  7. **strip()**：去除字符串首尾空白字符，中文的空格也会被移除

  8. **isEmpty()**：判断字符串是否为空

  9. **isBlank()**：判断字符串是否为空白字符串

  10. **replace()**：替换字符串中的字符

  11. **replaceAll()**：通过正则表达式替换所有符合规则的子串

  12. **split()**：分割字符串，传入正则表达式

  13. **join()**：使用指定字符串连接字符串数组

  14. format()静态方法，formatted()方法：格式化字符串，传入其他参数替换占位符，生成新的字符串

    • %s：显示字符串
    • %d：显示整数
    • %x：显示十六进制整数
    • %f：显示浮点数

  15. **String.valueOf()**：将其他类型数据转化为String

  16. **Boolean.parseBoolean()**：将字符串转为布尔值，转为其他类型类似

  17. **toCharArray()**：将字符串转换为char[]

  • String 和char在内存中总是以Unicode编码
  • 将byte[]转换为String已知编码可以直接转换，转换为byte[]时优先考虑UTF-8编码

StringBuilder

​ 高效拼接字符，StringBuilder是一个可变对象，可以预分配缓冲区，增加字符时不会创建新的临时对象。线程不安全

StringBuffer

​ 可变长字符串，运行效率慢，线程安全

StringJoiner

​ 方便拼接字符，可以指定拼接的模式，开头以及结尾

包装类

​ 对于八大基本类型定义的包装类：Boolean,Byte,Short,Integer,Long,Float,Double,Character。

包装类的创建

自动装箱，拆箱

​ java编译器会自动在int和integer之间转换

  • 装箱和拆箱发生在编译阶段，会影响代码的执行效率
  • 拆箱可以通过Number类实现，装箱可以通过包装类的方法实现
  • 自动装箱调用valueOf()方法，自动拆箱调用intValue()方法

不变类

​ 所有包装类型都是不变类，即不发生改变，比较包装类时要使用equals()

整数缓存区
  • 直接使用new 会构造新的对象，不同对象的地址不同，因此第一个返回false
  • 自动装箱时调用Integer.valueOf()方法，java初始情况下会创建一个整数包装类的数组，储存了-128127的整数包装类对象，如果在自动装箱的过程中数值在-128127之间，会直接返回初始化的包装类对象。超过这个范围，会新创建对象，因此第二个返回true
  • 200不在范围内，因此返回false

枚举类

​ 编译器可以自动检测某个值在枚举的集合内，并且不同用途的枚举需要不同的类型来标记

  • enum常量本身带有类型信息，编译器会自动检测类型错误

  • 不可能引用非枚举的值，无法通过编译

  • enum类型的每个常量在JVM中只有唯一一个实例，可以直接用==进行比较，不必一定使用equals()

  • 定义的enum类型无法被继承，无法使用new操作符创建enum，可以将enum用于switch语句

  • 每一个枚举的值都是class的实例，且是唯一的实例，他们有一些方法：

    1. name():返回常量名，无法被覆写
    2. ordinal():返回定义的常量顺序
    3. toString():返回和name()一样的字符串，但可以被覆写

记录类

  • 不变类具有一些特点：定义class时使用final，无法派生子类；每个字段使用final，保证创建实例后无法修改任何字段
  • 由于不变类的编写较为复杂，从java 14开始引入Record类，可以一行写出不变类。和enum类似，要通过record关键字由编译器实现继承

BigInteger

​ java.math.BigInteger可以表示任意大小的整数，BigInteger内部使用int[]数组来模拟非常大的整数。

​ BigInteger进行计算时必须使用实例方法，而且计算速度较慢。BigInteger可以转化为基本类型，如：byteValue(),longValue()等，超出范围会丢失高位信息。如果需要精确转换，需要使用byteValueExact()等方法。

BigDecimal

​ BigDecimal可以表示一个任意大小且精度完全准确的浮点数，是通过一个BigInteger和scale来表示的。使用**scale()表示小数位数，使用stripTrailingZeros()**可以将BigDecimal格式化为一个相等的去掉末尾0的BigDecimal。

  • 如果scale()为负数，则表示BigDecimal为整数，末尾有0
  • 可以使用**setScale()**对BigDecimal设置精度，可以四舍五入也可以截断
  • 进行加减乘时精度不会丢失，但在进行除法时，会有无法除尽的情况，因此需要指定精度和如何截断
  • divideAndRemainder():返回一个容量为2的数组，分别代表商和余数
  • 创建时尽量使用字符串创建，做运算时要使用特定的方法进行计算

比较BigDecimal

  • equals()：要求两个的scale()相等
  • 使用compareTo()：返回负数，正数和0，代表小于，大于和等于

Data

​ 大部分方法被Calendar类中的方法取代。表示特定的瞬间，精确到毫秒。

Calendar

​ 提供获取和设置各种日历字段的方法。构造方法为protected，不能直接创建对象，需要使用方法创建对象。

Simple DataFormat

​ 可以将日期转换为文本，或者将文本转换为日期。

System

​ 获取系统的属性数据和其他操作，构造方法私有。
